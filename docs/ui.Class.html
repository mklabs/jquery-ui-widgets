Class ui.Class

Based on jresig's implementaton (http://ejohn.org/blog/simple-javascript-inheritance/)
A base class which objects requiring oop implementation can extend. Class also handles the support of a _super method that allow you to override methods while keeping the ability to call overriden method somewhere in the process.
All subsequent controllers are inherited from BaseController which extends this Class.
