Module: controller

jQuery UI Controller Widget
A jQuery UI Widget that provides you clean and handy way to organize your jQuery code. Controllers organize event handlers through the power of event delegation. If something happens in your application, either it is browser event or custom ones, a controller should respond to it.
  • Controllers let you know where your code is!
  • Controllers force you to group events and label your html in specific ways.
  • Controllers are inheritable.
  • Controllers use event delegation.
